# Folder Ảnh Thương Hiệu Khác

Đặt tất cả ảnh sản phẩm của các thương hiệu khác (không phải Nike, Adidas, Puma, New Balance, Asics) vào folder này.

## Yêu cầu:
- Kích thước: 800x800px hoặc 1000x1000px
- Định dạng: JPG, PNG
- Nền: Trắng hoặc trong suốt
- Chất lượng: Độ phân giải cao
- Tên file: Viết thường, dùng dấu gạch ngang (-)
