## Yêu cầu ảnh:

1. **Định dạng**: JPG, PNG, WebP
2. **Kích thước đề xuất**: 
   - Ảnh chính: 800x800px hoặc 1000x1000px
   - Ảnh thumbnail: 400x400px
3. **Chất lượng**: Độ phân giải cao, nền trắng hoặc nền trong suốt
4. **Tên file**: 
   - Viết thường, dùng dấu gạch ngang (-)
   - Ví dụ: `nike-air-max-90.jpg`
   - Không dùng dấu cách hoặc ký tự đặc biệt

## Cách sử dụng:

1. Upload ảnh vào folder tương ứng với thương hiệu
2. Đặt tên file theo tên sản phẩm (viết thường, dùng dấu gạch ngang)
3. Trong admin panel, khi thêm/sửa sản phẩm, chọn ảnh từ folder này
4. Ảnh sẽ được tự động hiển thị trên website

## Lưu ý:

- Mỗi sản phẩm nên có ít nhất 1 ảnh chính
- Khuyến khích có nhiều góc chụp (trước, sau, bên, chi tiết)
- Ảnh nên được tối ưu hóa để tải nhanh (compress trước khi upload)
